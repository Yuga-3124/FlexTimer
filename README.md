# FlexTimer
超軽量・シンプル、そしてあなたの時間を大切にするためのタイマーアプリ。

## 概要
FlexTimerは、デスクトップでサッと動かせるWindows用のミニマルなタイマーです。余計な機能は一切なし。必要なときにすぐタイマーをセットして、集中時間を管理できます。

## 特徴
- ミニマルデザイン: 起動してすぐ使えるシンプルUI。

- 超高速起動: 軽量な .NET 10 アーキテクチャでサクサク動作。

## 使い方
最新のリリースからファイルをダウンロード。

好きなフォルダに展開。

FlexTimer.exe のショートカットをデスクトップに作成。

アイコンをクリックしてタイマー開始！

## 操作

右クリックでメニューが出せます

- START...設定したタイマーを開始します
- STOP...開始したタイマーを停止します
- NEWTIMER...新しいタイマーを設定します
- NOTIFICATION MESSAGE...タイマーが終了した際の通知で表示される言葉を設定できます(デフォルトは「時間になりました。」)
- HIDE 隠しますが時間になったら通知をします(再度表示させる際はタスクバーのアイコンをダブルクリック)
- EXIT このアプリを終了します

## 開発環境について
フレームワーク: .NET 10 (Windows Forms)

ビルド構成: Release (x64)
